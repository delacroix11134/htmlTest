CSS flexbox

CSS flexbox有以下幾種屬性

  • display
  • flex-direction
  • justify-content
  • align-items
  • align-self
  • align-content
  • flex-wrap
  • order
  • flex

display

display 是我們熟知的 CSS 屬性，對於 Flexbox 來說，多了兩種方式可以設定。
若設為flex則容器會變成flexbox，其內部的元素會多出許多排版上的彈性可以調整，並且容器之後的項目會自動換行。
另外一種屬性則是設為inline-flex，容器之後的項目不會換行，相當於inline-block的表現方式。

.flex{
display:flex;
}
.inline-flex{
display:inline-flex;
}

flex-direction

表示內容元素的排列方向

  • row(default) ：從左至右，從上到下
  • row-reverse ：與row相反
  • column ：從上到下，從左至右
  • column-reverse ：與column相反
.row {
flex-direction:row;
}
.row-reverse {
flex-direction:row-reverse;
}
.column {
flex-direction:column;
}
.column-reverse {
flex-direction:column-reverse;
}

justify-content

用來設定內容元素的水平對齊位置

  • flex-start(default) ：對齊左邊(main start)
  • flex-end ：對齊右邊(main end)
  • center ：置中對齊
  • space-between ：平均分配空間，左右元素會貼齊邊緣
  • space-around ：平均分配空間
.flex-start {
justify-content:flex-start;
}
.flex-end {
justify-content:flex-end;
}
.center {
justify-content:center;
}
.space-between {
justify-content:space-between;
}
.space-around {
justify-content:space-around;
}

align-items

設定內容元素的垂直對齊方式

  • flex-start(default) ：靠上對齊
  • flex-end ：靠下對齊
  • center ：置中對齊
  • stretch ：將元素全部撐開至容器的高度(*必須沒有設定元素的高)
  • baseline ：以內容元素的基線作為標準對齊
.flex-start{
align-items:flex-start;
}
.flex-end{
align-items:flex-end;
}
.center{
align-items:center;
}
.stretch{
align-items:stretch;
}
.baseline{
align-items:baseline;
}
.flex-item{
width:60px;
text-align:center;
}

align-self

設定與align-item相同，不同的是必須寫在內容元素上，用來覆蓋align-item屬性
如果只有其中一個元素使用baseline屬性，該元素沒有對位參考，因此會顯示靠上對齊

  • flex-start(default) ：靠上對齊
  • flex-end ：靠下對齊
  • center ：置中對齊
  • stretch ：將元素全部撐開至容器的高度
  • baseline ：以內容元素的基線作為標準對齊
.box1 {
align-self: flex-start;
line-height: 30px;
}

align-content

align-items 是針對內容為單行的元素進行處理，如果遇到多行的元素(需搭配flex-wrap使用)，就要使用 align-content 這個屬性，這個屬性總共有六個設定值。

  • flex-start(default) ：靠上對齊
  • flex-end ：靠下對齊
  • center ：置中對齊
  • stretch ：將元素全部撐開至容器的高度(*必須沒有設定元素高度)
  • space-between ：平均間距，但第一行與最後一行靠齊上下
  • space-around ：平均間距
.flex-start{
align-content:flex-start;
}
.flex-end{
align-content:flex-end;
}
.center{
align-content:center;
}
.space-between{
align-content:space-between;
}
.space-around{
align-content:space-around;
}
.stretch{
align-content:stretch;
}

flex-wrap

設定是否讓內容元素自動換行

  • nowrap(default) ：單行
  • wrap ：多行(自動換行)
  • wrap-reverse ：多行，但是順序反轉
.nowrap {
flex-wrap: nowrap;
}
.wrap {
flex-wrap: wrap;
}
.wrap-reverse {
flex-wrap: wrap-reverse;
}
.nowrap-column {
flex-direction: column;
flex-wrap: nowrap;
}
.wrap-column {
flex-direction: column;
flex-wrap: wrap;
}
.wrap-reverse-column {
flex-direction: column;
flex-wrap: wrap-reverse;
}

flex

由三個屬性組成：flex-grow, flex-shrink, flex-basis 如果只填一個數字(無單位)預設以flex-grow呈現

  • flex-grow ：數字無單位，當元素的flex-basis小於他在容器中分配到的長度則會照數字的比例分配空間。預設0，不作彈性變化，設為1以上則會自動分配(不能為負值)
  • flex-shrink ：數字無單位，當元素的flex-basis大於他在容器中分配到的長度則會照數字的比例分配空間。預設1，設為0則不作彈性變化(不能為負值)
  • flex-basis ：子元素的基本大小，作為父元素的大小比較基準，預設值為 0，也因為預設值為 0，所以沒有設定此屬性的時候，會直接採用 flex-grow 屬性，flex-basis 也可以設為 auto，如果設為 auto，就表示子元素以自己的基本大小為單位。
.box1 {
flex: 1 2 200px;
}
.box2 {
flex: 2 1 100px;
}
