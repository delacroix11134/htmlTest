CSS垂直置中的幾個方法

設定行高:只有文字被置中

單行

只能讓文字置中 box會出問題

.box {
line-height:150px;
display: inline-block;
}
.content {
display: inline-block;
}

加入僞元素::before

可以多行文字、box置中
缺點程式碼行數較多 不易閱讀

.content {
display:inline-block;
vertical-align: middle;
}
.box::before {
content: '';
display: inline-block;
vertical-align: middle;
height: 100%;
}

使用flexbox

超級簡單

.box {
display: flex;
justify-content: center;
align-items: center;
}

使用position:absolute

缺點absolute元素會互相重疊
在物件多的時候要小心

.box {
position: relative;
}
.content {
position: absolute;
top: 0;
right: 0;
bottom: 0;
left: 0;
margin: auto;
}
